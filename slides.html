Margin

Margin adds space outside and around an element and also uses 4 values just like padding.

Using the example below, let's change some values to see the difference between margin and padding. Try using 1 - 4 values.

padding and margin test

Center aligning with margin

margin can also be used to auto center align block level elements.

First, a width needs to be set. Then, setting the left & right values to auto will find the center of the page. The 0 just refers to the top and bottom value.

Gradients and Background Images

In the Skills section, the text may be hard to read depending on the background image. We can actually use multiple background images to set a gradient on top of the image!

Declare the gradient first so it can sit on top of the image. We also want to see the image through the gradient so use rgba color values to set the alpha transparency. Separate multiple backgrounds with a comma.
